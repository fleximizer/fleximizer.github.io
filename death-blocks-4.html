Death Blocks 4

“Death Blocks 4” is a FPS to shoot looming zombies.

So many features were added!
The place you dispatched has occupied by dreadful zombies.
The longer you stay, the more danger zombies appear…
Blow’em all and survive!

Features

Minecraft like voxel based graphics.
Various weapon like Machingun, Shotgun, Deagle, Grenade, RPG, Knife, AR.
Put the blocks and create your own building
Boss battles
Several types of unique zombies will be appeared.
Upgrade the weapon and armor by gem you got.
Custom map edit mode
Riding big robots

*Multiplayer mode will be added in a later update.
